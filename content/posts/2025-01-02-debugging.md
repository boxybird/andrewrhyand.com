# Debugging

## The Process

1. Reproduce the bug
2. Isolate the cause
3. Fix it
4. Verify the fix
5. Add a test

## Tools I Use

- `dd()` and `dump()`
- Laravel Telescope
- Xdebug with step debugging
- `git bisect` for regressions

## The Golden Rule

If you can't reproduce it, you can't fix it.
